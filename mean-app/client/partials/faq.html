Πιο συχνές ερωτήσεις και απαντήσεις

  • Ποιος είναι ο σκοπός αυτής της ιστοσελίδας?

    • Ο βασικός σκοπός είναι η δημιουργία μίας διαδικτυακής πλατφόρμας μέσω της οποίας θα διευκολύνονται οι γονείς να επιλέγουν δραστηριότητες για τα παιδιά τους με βάση διάφορα κριτήρια, όπως η ηλικία, το είδος της δραστηριότητας, το κόστος της κ.ά.. Επιπλέον, αποσκοπούμε στη προώθηση και ανάδειξη των παρόχων και των υπηρεσιών τους.

  • Χρειάζεται να πληρώσω?

    • Όχι, εκτός αν θέλετε να κάνετε δωρεά.

  • Τι μπορώ να κάνω χωρίς λογαριασμό?

    • Μπορείτε μόνο να ψάξετε για δραστηριότητες. Δεν μπορείτε να αγοράσετε και να κάνετε κρατήσεις, δεν μπορείτε να αποθηκεύσετε τις δραστηριότητες που ενδεχομένως να σας ενδιαφέρουν ούτε να κάνετε προσαρμοσμένη αναζήτηση των δρασστηριοτήτων με βάση την τοποθεσία.

  • Δεν μπορώ να συνδεθώ.

    • Σε περίπτωση ου ξεχάσετε τον κωδικό σας πατήστε στον σύνδεσμο κατά την σύνδεση σας στον λογαριασμό και ένας admin μπορεί να αλλάξει τον κωδικό σας(reset password).

  • Τι μπορώ να κάνω μέσω του προφίλ μου.

    • Στην περίπτωση του χρήστη γονέα :
      • Μπορείς να δεις και να επεξεργαστείς ορισμένα δεδομένα του λογαριασμού σου.
      • Μπορείς να προσθέσεις πόντους στο πορτοφόλι σου μέσω κάρτας.
      • Μπορείς να καθορίσεις την τοποθεσία σου ώστε να μπορείς να κάνεις αναζητήσεις με κριτήριο την τοποθεσία.
    • Στην περίπτωση του χρήστη παρόχου :
      • Μπορείς να δεις και να επεξεργαστείς ορισμένα δεδομένα του λογαριασμού σου.
      • Μπορείς να δεις κάποια χρήσιμα στατιστικά που αφορούν τον λογαριασμό σου.
      • Μπορείς να δεις το ιστορικό των δραστηριοτήτων σου.
      • Μπορείς να αλλάξεις την περιγραφή των υπηρεσίων σου που θα φαίνεται στο δημόσιο προφίλ σου.
      • Μπορείς να καθορίσεις την τοποθεσία σου ώστε να μπορείς να κάνεις αναζητήσεις με κριτήριο την τοποθεσία.

  • Είμαι πάροχος και δεν μπορώ να επιτελέσω καμία λειτουργία.

    • Στην περίπτωση που είστε πάροχος και μόλις έχετε εγγραφεί θα πρέπει να περιμένετε ένα διάστημα μέσα στο οποίο θα γίνουν κάπιοι έλεγχοι και στην περίπτωση που εγκριθείτε θα μπορείτε να επιτελέσετε όλες τις επιθυμητές λειτουργίες αλλιώς δεν σας παρέχετε καμία δυνατότητα.
